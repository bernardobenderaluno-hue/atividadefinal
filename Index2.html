Catálogo de Entregas

Descrição: Desenvolvimento de um script automatizado e modular em Python para mapear, validar e catalogar as principais competições esportivas (como o EFIPAN) para o portal da cidade, gerando um banco de dados dinâmico.

3. Otimização de Rota Logística de Caminhão

Eng. de Produção Lógica de Programação

Descrição: Implementação do algoritmo do "vizinho mais próximo" em Python para otimizar pontos de entrega num plano cartesiano, calculando consumo de diesel e necessidade de reabastecimento durante a rota.

Ver Código no GitHub

4. Análise de Tensões em Treliças Planas

Eng. Mecânica/Civil Matplotlib

Descrição: Script computacional que recebe dados estruturais dinâmicos, calcula tensões normais, classifica esforços (tração/compressão), avalia o Fator de Segurança (FS) contra escoamento e plota gráficos comparativos.

Ver Código e Gráficos

5. Viabilidade de Sistema Solar Fotovoltaico

Eng. Elétrica Eng. Econômica Matplotlib

Descrição: Solução completa de engenharia para dimensionamento de uma planta solar fotovoltaica. Inclui análise de consumo versus geração com base em HSP e simulação do Fluxo de Caixa Acumulado (VPL) em um horizonte de 25 anos.
